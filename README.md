# Pizza Passport (The Sequel)
Pizza Passport application made using sequelize

Allows users log and track the pizzas they wish to eat. 


**Restaurant customer feature coming soon.**

- Javascript

- mySQL
- Sequelize 
- Heroku (for deployment)
